# 小曦薇

开发者：Anbunengsi

这是一个 Windows 单文件角色互动项目，以公开可见的田曦薇活动素材为造型参考，白色短裙造型，并制作成写实风格的高清动画角色。

## 当前功能

- 写实全身 4K 源素材，运行时可自由缩放。
- 所有循环动作由关键姿势和双向运动网格实时形变到至少 24 个显示阶段；每个阶段只绘制一个不透明人物，避免透明双人重影。动作总时长与 v3.0.1 保持一致。
- 鼠标仅在角色附近时驱动眼神和轻微头部跟随；光标离开近距离范围后自动回到仅呼吸眨眼的普通待机。
- 鼠标真实拖动超过容差后会先“吓一跳”，拖动期间保持受惊姿势；松手后立即皱眉、抬脚、跺脚并抱臂生气。
- 普通单击、双击挥手、睡眠唤醒和玩手机起身均有独立判定，不会被鼠标轻微抖动误触发。
- 公共随机待机包括手势舞、唱歌表演、女侠演戏、飞吻互动、坐下玩手机和侧躺睡觉；内置 vivo 白裙另有“卖萌”“哈哈哈大笑”“哭一下”和“白裙星光亮相”四套动作。
- 坐下后会持续刷手机；侧躺后会持续睡眠，并从头部上方飘出 `z/Z`。两种状态均由单击结束。
- 切换皮肤时只对人物 alpha 轮廓做切片伪 3D 旋身，并在旋身中由头到脚更新造型，最后定格 pose 后平滑回到待机；不会翻转整个窗口或画布。
- 随增强版附带两套完整 176 帧外置皮肤：“临安公主｜大奉打更人”和“黄橙子｜如此可爱的我们”。临安拥有手动专属持续动作“公主荡秋千”：人物正面居中、视线朝前，循环严格按“c2 低点 A → c3 向前高点 → c4 低点 B → c5 向后高点 → c2 低点 A”沿画面纵深前后交替；已修复旧循环边界连续两次向前的问题，并新增 c5→c2 单主体运动网格。选中后会一直摆动，单击人物会等待 c2 低点，再进入 c6–c7 下秋千。黄橙子的“橙子放学元气招呼”仍可进入菜单与随机待机。
- 可选的田曦薇微博新动态提醒；首次检查只建立基线，不弹出历史内容。
- 右键菜单可预览动作、开关微博提醒、立即检查微博和退出程序。

## 文件夹

- `releases/历史版本`：v1.0 标准版、v2.0 4K 版以及 v3.0–v3.0.5 增强版。
- `releases/最新版`：本次增强版 EXE。
- `source`：可复现构建所需的源码、脚本和动作源图。
- `qa`：动画联系表、GIF 与构建验证报告。
- `docs`：使用说明、动作设计依据和免责声明。

## 快速开始

进入 `releases/最新版`，运行 `小曦薇.exe`。请保留 EXE 同级的 `skins` 文件夹；右键小曦薇并打开“皮肤”菜单，即可在内置 vivo 白裙、临安公主和黄橙子之间切换。

从 GitHub 获取时，请在仓库的 **Releases** 页面下载 `小曦薇-完整包.zip`，解压后直接运行 `小曦薇.exe`。源码仓库不会提交用户参考照片、生成中间图、QA 缓存或大型二进制文件。

换肤时会播放“人物 alpha 切片伪 3D 旋身 → 从头到脚变装 → 定格 pose → 回到待机”的过渡。程序本体为单文件 EXE；两套可选皮肤因包含 352 张高清透明帧，独立放在 `skins` 文件夹中，便于后续继续扩展。
