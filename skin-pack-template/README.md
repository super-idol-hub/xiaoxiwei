# 皮肤包模板

1. 把此文件夹复制到 Windows 互动角色 EXE 同级的 `skins` 文件夹。
2. 把文件夹改成安全的皮肤 ID，例如 `summer-blue`。
3. 同步修改 `skin.xml` 中的 `id`、`name` 和 `developer`。如需在 v3.0.6 启用该皮肤自己的动作，把 `exclusiveAction` 填成动作名；不需要时保持空字符串或删去该属性。
4. 放入完整的 `frames.zip`；压缩包内部直接使用 `r00/c00.png` 形式的路径。
5. 启动 Windows 互动角色，在右键“皮肤”菜单中切换并逐动作检查。

每张 PNG 固定为 `528×808` RGBA；单条目不超过 32 MiB，压缩包及解压后总量分别不超过 128 MiB。完整皮肤必须恰好包含 176 张关键帧，不能加入额外文件。

接口仍为 `apiVersion="1"`。`exclusiveAction` 非空时，r15 必须包含 8 张属于该皮肤、姿势连续的专属动作关键帧，该动作会出现在当前皮肤的动作菜单；普通皮肤也可参与随机待机。官方 `linan-princess` 的持续秋千属于运行时特例，只能手动触发，不能通过更改动作名称复制给其他皮肤。留空时 r15 作为兼容占位，不会触发。r16–r17 保留，r18–r23 是所有皮肤共享的公共待机动作，所有行仍必须完整提供。

完整行映射与校验规则见 `docs/皮肤包接口.md`。
