# 小曦薇 v3.0.4

本版本将所有对外产品名称统一为“**小曦薇**”。

- 主程序文件：`小曦薇.exe`
- 完整包：`小曦薇-完整包.zip`
- 内置 vivo 白裙，以及临安公主、黄橙子两套可选皮肤
- 近距离鼠标视线跟随；光标离开后回到呼吸眨眼待机
- 内置卖萌、哈哈大笑、哭等动作，两套扩展皮肤各有专属动作
- 人物轮廓伪 3D 旋身换装过渡
- 176 张 4K 关键帧，自检通过
