# 小曦薇 v3.0.6

本版本把临安公主专属动作重做为可持续互动的“公主荡秋千”。

- 主程序文件：`小曦薇.exe`
- 完整包：`小曦薇-完整包.zip`
- 内置 vivo 白裙，以及临安公主、黄橙子两套可选皮肤
- 近距离鼠标视线跟随；光标离开后回到呼吸眨眼待机
- 白裙飞吻 8 帧统一为稳定的深黑棕发色
- 内置白裙新增“白裙星光亮相”专属动作
- 临安公主保持原红色皮肤，新增剧中同款双绳雕花长椅秋千；秋千沿画面纵深前后摆动，不做左右横移
- 荡秋千时人物与视线始终朝正前方，只保留随摆动产生的轻微抬头、低头和眨眼，不再左右回头
- “公主荡秋千”只从动作菜单手动进入；循环严格按“c2 低点 A → c3 向前高点 → c4 低点 B → c5 向后高点 → c2 低点 A”前后交替；已修复旧循环边界连续两次向前的问题，并新增 c5→c2 单主体运动网格
- 上秋千后会持续摆动；单击人物会等待 c2 低点，再进入 c6–c7 下秋千
- 内置卖萌、哈哈大笑、哭等动作；两套扩展皮肤仍各有专属动作
- 人物轮廓伪 3D 旋身换装过渡
- 176 张 4K 关键帧；秋千进入、循环、退出各至少 24 个显示阶段，自检通过
